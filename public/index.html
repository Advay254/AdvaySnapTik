For developers

Clean REST API

Integrate TikTok downloading into apps, n8n workflows, and automation — one authenticated GET request.

GET /api/v1/info GET /api/v1/download JSON response n8n ready RapidAPI
Get API Key on RapidAPI →
# Get all download URLs
GET /api/v1/info?url={tiktok_url}
X-API-Key: your_key

# Stream no-watermark video
GET /api/v1/download
    ?url={url}&type=video

# Stream MP3 audio
    &type=audio

# Slideshow image at index N
    &type=image&index=0
