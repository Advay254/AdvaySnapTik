# public/og-image.png — Open Graph Image Placeholder

This file is referenced in index.html for social sharing previews on Twitter, Facebook, WhatsApp, etc.

## Required File
- **Filename:** `og-image.png`
- **Location:** `public/og-image.png`
- **Size:** 1200×630px (standard OG image)
- **Format:** PNG

---

## Image Prompt

Use this prompt in Ideogram, Midjourney, or DALL·E 3:

```
A sleek social media preview card, exactly 1200x630px.
Solid near-black background #0d0d0d.

Left half:
- Large bold headline in white sans-serif: "AdvaySnapTik"
  where "Snap" is colored teal #25f4ee and "Tik" is red #fe2c55
- Below, smaller white text: "Download TikTok Videos Without Watermark"
- Even smaller gray text: "Free · No watermark · No login needed"

Right half:
- A stylized abstract phone mockup outline (no screen content)
- A glowing downward arrow icon — teal #25f4ee top half, red #fe2c55 bottom half
- Subtle radial glow aura behind the arrow

Background effects:
- Subtle radial glow: teal at top-left corner, red at bottom-right corner
- No gradients on text — pure white text only
- Minimal, modern, no stock photos, no people

Output: 1200×630px PNG, horizontal (landscape)
```

## After Generating

1. Resize to exactly **1200×630px** using [Squoosh](https://squoosh.app) or any editor
2. Save as `og-image.png`
3. Place at: `public/og-image.png`

That's it — the meta tags in `index.html` already reference this path.
